# PULSO - Plataforma de Radio White-Label

Plataforma de radio white-label para Radio Bienvenida 93.3 FM.

## ⚠️ Seguridad - Solo usar pnpm

**IMPORTANTE:** Este proyecto solo debe usar `pnpm` para la gestión de dependencias. No usar `npm install` debido a riesgos de seguridad en repositorios públicos.

## Getting Started

Primero, instala las dependencias con pnpm:

```bash
pnpm install
```

Luego, ejecuta el servidor de desarrollo:

```bash
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) con tu navegador para ver el resultado.

## Scripts Disponibles

- `pnpm dev` - Inicia el servidor de desarrollo
- `pnpm build` - Construye la aplicación para producción
- `pnpm start` - Inicia el servidor de producción
- `pnpm lint` - Ejecuta el linter

## Características

- Reproductor de audio en vivo
- Transmisión de video HLS
- Animación de bienvenida premium
- Logo animado con Canvas API
- Navegación simplificada
- Espacios publicitarios premium
- Diseño responsivo móvil
